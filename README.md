#  Classificador de Nível do Herói

Projeto simples em JavaScript que classifica o nível de um herói com base na quantidade de XP.

##  Tecnologias
- JavaScript

##  Conceitos praticados
- Variáveis
- Estruturas condicionais (if / else if)
- Operadores relacionais

##  Lógica aplicada
O sistema classifica o herói conforme a seguinte regra:

- Ferro: XP < 1000
- Bronze: 1001–2000
- Prata: 2001–5000
- Ouro: 5001–7000
- Platina: 7001–8000
- Ascendente: 8001–9000
- Imortal: 9001–10000
- Radiante: >= 10001
